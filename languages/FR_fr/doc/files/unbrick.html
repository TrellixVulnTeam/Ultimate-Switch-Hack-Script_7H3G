Infos sur le script de débrickage

 

Ceci est un script batch automatisant le débrickage de la Switch pour la plupart des problèmes software (méthode ChoiDuJour grandement automatisée et optimisée, voir cette page qui explique la méthode manuelle de débrickage via ChoiDuJour).

Attention, une fois le script de TegraExplorer exécuté, toutes les données de la sysnand seront supprimées. Veuillez faire un dump complet de la nand avant tout (même si la console ne fonctionne plus) via Hekate ou NXNandManager par exemple (non couvert par ce projet) et veuillez également faire un dump des clés de la console via Lockpick-RCM (si celui-ci ne fonctionne pas il est probable que cette procédure ne donne rien). Il est également préférable d'utiliser une SD vierge (1 GO suffit largement), ceci dit cela n'est pas obligatoire mais si certains fichiers de votre SD sont différents de ceux de cette procédure ils seront remplacés, notamment le fichier "bootloader\hekate_ipl.ini" qui est le plus important si vous utilisez Hekate et Atmosphere sera mis à jour vers la version incluse dans cette procédure (Atmosphere 0.12.0 actuellement).

 

Fonctionnalités

 
  • Création du package 6.1.0 grâce à ChoiDuJour.
  • Installation des drivers du mode RCM (driver "APX").
  • Téléchargement d'un firmware choisi par l'utilisateur et préparation de la SD avec celui-ci pour ChoiDuJourNX, le homebrew est également copié durant le processus.
  • Préparation d'une SD, formatage (FAT32 ou EXFAT) et copie des fichiers nécessaires au débrickage sur celle-ci.
  • Lancement des payloads nécessaires tout au long du script via le mode RCM de la Switch.
  • Restauration du package créé par ChoiDuJour grâce à un script exécutable via le payload TegraExplorer (ne pas utiliser ce script hors de la procédure de ce projet car cela pourrait poser des problèmes sur la console, résolvables ceci dit en faisant la procédure depuis le début).
  • Réécriture d'un fichier BOOT0 avec les keyblobs obtenu par Linkle via le fichier de clés dumpé sur la console ciblée par cette modification avec Lockpick-RCM.
